## 3. 레이아웃 및 타이포그래피

### 3.1 게시판 목록 레이아웃

- **상단 고정 헤더 (Sticky Header)**: 스크롤 시에도 테이블의 `thead`가 `top: 52px`에 고정되어야 함.
- **테이블 스타일**: `border-collapse: separate;`와 `border-spacing: 0;`을 사용하여 둥근 모서리 유지.

### 3.2 게시글 상세 레이아웃

- **2컬럼 그리드 (관리자)**: 댓글 섹션(70%)과 내부 메모 섹션(30%)이 가로로 나열됨.
- **1컬럼 레이아웃 (담당자)**: 내부 메모 섹션이 숨겨지고 댓글 섹션이 100% 가로폭 사용.
