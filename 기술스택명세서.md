# 기술 스택 명세서 (v1.0)

본 문서는 개발자가 시스템 환경을 빠르게 파악하고 유지보수할 수 있도록 기술 사양을 정리합니다.

## 1. Core Frameworks

- **Frontend**: Next.js 14+ (App Router)
- **Language**: TypeScript (Strict Mode)
- **State Management**: Zustand
  - `persist` 미들웨어: 사용자 세션 및 로그인 정보 로컬 스토리지 유지.
  - `devtools`: 개발 단계 디버깅 지원.

## 2. UI & Styling

- **CSS**: Pure CSS (CSS Variables 기반)
  - `src/styles/base.css`: 전역 변수 및 초기화.
  - `src/styles/board.css`: 게시판 주요 컴포넌트 스타일.
- **Icons**: Lucide React (추천 사양, 현재 SVG 직접 사용 중).
- **Editor**: Lexical Editor (Meta에서 제공하는 확장 가능한 텍스트 에디터).

## 3. 주요 라이브러리 목록

| 라이브러리 | 용도                  | 비고                        |
| :--------- | :-------------------- | :-------------------------- |
| `date-fns` | 날짜 포맷팅 및 비교   | `yyyy-MM-dd` 형식 표준 사용 |
| `lexical`  | Rich Text 에디터 구현 | 템플릿 삽입 플러그인 커스텀 |
| `zustand`  | 전역 상태 관리        | 게시글 데이터 및 세션 관리  |

## 4. 폴더 구조 (Project Structure)

```text
src/
├── app/               # Next.js App Router (Page & Layout)
├── components/        # 공통 UI 및 게시판 전용 컴포넌트
│   ├── ui/            # 버튼, 배지 등 아토믹 컴포넌트
│   └── board/         # 목록, 아이템 등 도메인 컴포넌트
├── store/             # Zustand Store (비즈니스 로직)
├── styles/            # CSS 파일 관리
└── lib/               # 공통 타입 및 모킹 데이터
```
