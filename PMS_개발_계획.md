# 게시판형 PMS 개발 계획

## 📋 기능 요구사항 분석

### 백엔드 기능 (목킹데이터 대상)
1. **비밀게시글(Secret) 기능** - 작성자와 관리자만 열람 가능
2. **게시글 처리 상태(Status)** - 접수, 확인, 진행, 완료, 보류, 취소 등
3. **태깅 시스템** - 고객사별 프로젝트별 코드 태깅
4. **키워드 검색** - 제목+내용+댓글+작성자+기간 검색
5. **댓글 시스템** - 댓글 검색, 내부 메모 기능 (관리자 전용)
6. **페이징 처리** - 무한스크롤 구현
7. **Lexical 에디터** - WYSIWYG 에디터
8. **멀티 파일 업로드** - 드래그앤드롭, 다양한 포맷 지원
9. **유효성 검사** - 필수 필드 검증
10. **작업 공수 기록** - 시간 기록 (관리자 기능)

## 🚀 개발 순서 및 계획

### Phase 1: 기본 구조 및 목킹데이터 구축
1. **프로젝트 초기 설정**
   - Next.js 프로젝트 구조 확인 및 의존성 설치
   - TypeScript 인터페이스 정의
   - 목킹데이터 구조 설계

2. **목킹데이터 생성**
   - 게시글 데이터 구조 정의
   - 사용자/권한 데이터 생성
   - 댓글 및 내부 메모 데이터
   - 파일 업로드 데이터 구조

3. **상태 관리 시스템 구축**
   - 게시글 상태 관리 (Redux Toolkit 또는 Zustand)
   - 검색 및 필터링 상태
   - 사용자 권한 상태

### Phase 2: 코어 컴포넌트 개발
4. **게시글 목록 페이지**
   - 게시글 리스트 컴포넌트
   - 상태 배지 표시
   - 검색 및 필터링 UI
   - 페이징/무한스크롤 구현

5. **게시글 상세 페이지**
   - 게시글 내용 표시
   - 댓글 컴포넌트
   - 내부 메모 기능 (관리자 전용)

6. **글쓰기 페이지**
   - Lexical 에디터 통합
   - 파일 업로드 컴포넌트
   - 유효성 검사 구현

### Phase 3: 고급 기능 및 마무리
7. **관리자 기능**
   - 작업 공수 입력
   - 내부 메모 관리
   - 게시글 상태 변경

8. **보안 및 권한**
   - 비밀게시글 접근 제어
   - 사용자 권한 검증

9. **UI/UX 개선**
   - 반응형 디자인
   - 로딩 상태 및 에러 처리
   - 사용자 피드백

## 📁 파일 구조 계획

```
src/
├── app/
│   ├── board/
│   │   ├── page.tsx              # 게시글 목록
│   │   ├── [id]/
│   │   │   └── page.tsx          # 게시글 상세
│   │   └── write/
│   │       └── page.tsx          # 글쓰기
│   └── admin/
│       └── page.tsx              # 관리자 페이지
├── components/
│   ├── board/
│   │   ├── BoardList.tsx         # 게시글 리스트
│   │   ├── BoardItem.tsx         # 게시글 아이템
│   │   ├── BoardDetail.tsx       # 게시글 상세
│   │   ├── CommentSection.tsx    # 댓글 섹션
│   │   ├── Editor.tsx            # Lexical 에디터
│   │   └── FileUpload.tsx        # 파일 업로드
│   └── ui/                       # 공통 UI 컴포넌트
├── lib/
│   ├── mockData.ts               # 목킹데이터
│   ├── types.ts                  # TypeScript 타입
│   └── utils.ts                  # 유틸리티 함수
└── store/
    ├── boardSlice.ts             # 게시글 상태 관리
    └── userSlice.ts              # 사용자 상태 관리
```

## 🔧 기술 스택
- **Frontend**: Next.js 14+, TypeScript, Tailwind CSS
- **State Management**: Zustand 또는 Redux Toolkit
- **Editor**: Lexical (WYSIWYG)
- **File Upload**: React Dropzone
- **Icons**: Lucide React
- **Mocking**: 목킹데이터 (JSON 기반)
