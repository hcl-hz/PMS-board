# 기능 명세서 (v1.0)

본 문서는 기획자와 개발자가 구현해야 할 PMS 시스템의 상세 동작을 정의합니다.

## 1. 게시판 목록 (Board List)

### 1.1 무한 스크롤 (Infinite Scroll)

- **동작**: 페이지 하단 도달 시 `IntersectionObserver`를 통해 다음 20개의 게시물을 동적으로 로드함.
- **예시 코드 (Client-side)**:

```typescript
const observer = new IntersectionObserver((entries) => {
  if (entries[0].isIntersecting && hasMore) {
    loadNextPage();
  }
});
```

### 1.2 통합 검색 및 필터

- **검색 범위**: 제목, 내용, 작성자 이름, 태그 이름, 댓글 작성자 이름.
- **기간 검색**: `YYYY-MM-DD` 형식의 시작일과 종료일을 포함하는 게시물 필터링.
- **프로젝트 정렬**: 프로젝트 목록은 가나다순(한글 우선) 정렬.

## 2. 게시글 작성 및 수정 (Write & Edit)

### 2.1 에디터 및 템플릿

- **서식 지원**: Lexical WYSIWYG 에디터 활용 (Bold, Italic, Underline, Strikethrough).
- **템플릿 삽입**: '템플릿 삽입' 버튼 클릭 시 본문에 사전 정의된 서식(내용, 요청사항 등) 강제 삽입.

### 2.2 파일 업로드 보안

- **제한사항**: `.exe`, `.bat`, `.sh` 등 실행 파일 업로드 시 즉시 차단 메시지 출력.
- **용량 제한**: [지정 필요] (현재 프론트엔드에서는 제한 없음).

## 3. 게시글 상세 (Detail View)

### 3.1 상태 변경 프로세스

- **관리자 전용**: 상태 배지 클릭 시 드롭다운 노출 -> 상태 선택 -> **확인 모달(`변경`)** -> 상태 업데이트.
- **동작**: 상태 변경 시 `isNotice` 플래그 자동 업데이트 (상태가 '공지'이면 `true`).

### 3.2 작업공수 관리

- **입력**: 관리자가 숫자 입력 시 즉시 저장.
- **단위**: 시간(Hour) 단위로 표시.

## 4. 댓글 및 내부 메모

### 4.1 글자 수 제한

- **제한**: 최대 200자.
- **UI**: 현재 글자 수 실시간 표시 (`15/200`).
- **입력**: `Ctrl + Enter`를 통한 빠른 등록 지원.
